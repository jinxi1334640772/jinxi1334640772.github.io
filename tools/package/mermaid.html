📊 Mermaid 使用手册

💡 简介: Mermaid 是一个基于 JavaScript 的图表和流程图生成库，可以用简单的文本语法创建复杂的图表。本手册提供所有图表类型的完整示例。

📋 目录 ​

  • 1. 流程图 (Flowchart)
  • 2. 序列图 (Sequence Diagram)

  • 4. 甘特图 (Gantt Chart)
  • 5. 用户旅程图 (User Journey)
  • 6. Git 图 (Git Graph)
  • 7. 实体关系图 (Entity Relationship Diagram)

1. 流程图 (Flowchart) ​

1.1 基础流程图 ​

代码示例：

text
```mermaid
flowchart TD
    A[开始] --> B{是否登录?}
    B -->|是| C[进入首页]
    B -->|否| D[跳转登录页]
    D --> E[输入用户名密码]
    E --> F{验证通过?}
    F -->|是| C
    F -->|否| G[提示错误信息]
    G --> E
    C --> H[结束]
```

🔧 高级技巧 ​

1. 子图 (Subgraph) ​

代码示例：

text
```mermaid
flowchart TD
    subgraph 前端系统
        A[Vue.js] --> B[Router]
        B --> C[Vuex]
        C --> D[Components]
    end
    
    subgraph 后端系统
        E[Node.js] --> F[Express]
        F --> G[中间件]
        G --> H[API]
    end
    
    subgraph 数据库
        I[MySQL] --> J[数据表]
```

  • 避免过长的标签文本
  • 使用表情符号增强可读性

  • 合理选择图表方向
  • 避免过于复杂的关系
  • 使用子图组织相关内容

  • 限制节点数量（建议不超过30个）
  • 避免过深的嵌套关系
  • 使用简化的语法
