hash路由 ​

hash路由在html5之前，是解决SPA应用路由跳转问题采用的解决方案。hash变化不会向服务器重新发起请求，并且服务器也无法获取到hash值，所以不会触发页面的重新渲染。前端可以通过监听hashchange事件监听hash值的变化。

js
//监听hash变化
window.addEventListener('hashchange',function(){
  // hash变化后触发hashchange事件，
  // 这里做相应操作
})
